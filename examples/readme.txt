================================================================================
  一、软件包概述
================================================================================

本软件包实现了基于符号回归的材料发现工作流，结合了：
  • SISSO（符号回归）- 生成可解释的数学公式
  • MACE（机器学习势函数）- 高效能量计算
  • 主动学习 - 智能选择候选材料

================================================================================
  四、系统要求
================================================================================

硬件：
  • CPU: 多核处理器（推荐 8 核以上）
  • 内存：至少 16GB RAM（推荐 32GB）
  • 存储：至少 10GB 可用空间
  • GPU: 可选（用于加速 MACE 计算）

软件：
  • 操作系统：Linux, macOS, Windows (WSL2)
  • Python: 3.9 - 3.11
  • Conda: 推荐用于依赖管理

================================================================================
  六、配置说明
================================================================================

主要参数（在 workflow_active_learning_mace_fixed.py 中）：

  AL_N_ITERATIONS = 30        # 主动学习迭代次数
  AL_BATCH_SIZE = 5           # 每批选择的材料数
  AL_UCB_KAPPA = 0.1          # UCB 探索参数
  
  MACE_RELAX = False          # 是否进行结构弛豫
  MACE_DEVICE = 'cpu'         # 计算设备：'cpu' 或 'cuda'
  
  SISSO n_dim = 2             # 描述符维度
  SISSO max_rung = 2          # 公式层级

快速测试配置：
  AL_N_ITERATIONS = 5
  AL_BATCH_SIZE = 3
  MACE_RELAX = False

================================================================================
  七、输出结果
================================================================================

运行完成后，输出目录包含：

  workflow_output_al_mace/
  ├── model_info.json                 - 模型性能信息
  ├── active_learning_history.csv     - 主动学习历史
  ├── sisso_initial/                  - 初始 SISSO 模型
  ├── sisso_final/                    - 最终 SISSO 模型
  └── sisso_al/iter_*/                - 各迭代模型

关键指标：
  • R² (决定系数) - 越接近 1 越好
  • RMSE (均方根误差) - 越小越好，单位 eV/atom
  • SISSO 公式 - 可解释的描述符

================================================================================
  八、典型性能
================================================================================

计算时间（参考）：
  • 快速测试：~10 分钟（5 次迭代，无弛豫）
  • 标准配置：~1 小时（30 次迭代，无弛豫）
  • 高精度：~5-10 小时（30 次迭代，有弛豫）

模型性能（典型）：
  • 初始 R²: ~0.71, RMSE: ~0.60 eV/atom
  • 最终 R²: ~0.75, RMSE: ~0.56 eV/atom
  • 改进：R² +5.6%, RMSE -6.7%

================================================================================
  九、常见问题
================================================================================

Q: conda 未找到？
A: 先安装 Miniconda: https://docs.conda.io/en/latest/miniconda.html

Q: sissopp 导入失败？
A: 运行：conda install -c conda-forge sissopp -y

Q: 文件找不到？
A: 确保在软件包目录运行，或检查路径配置

Q: 内存不足？
A: 减少 AL_BATCH_SIZE 或使用更小的数据集

Q: 计算太慢？
A: 关闭 MACE 弛豫：MACE_RELAX = False
